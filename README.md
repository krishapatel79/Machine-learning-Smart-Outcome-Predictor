# Smart Outcome Predictor

An Ensemble Learning project that predicts **student course completion** (Classification) and **final performance score** (Regression) using multiple ensemble machine learning algorithms.

---

## Project Description

The Smart Outcome Predictor is designed for an EdTech platform to analyze student data and predict academic outcomes. The project compares different ensemble learning techniques such as Bagging, Boosting, Voting, and Stacking to identify the most accurate and reliable model.

---

## Objectives

- Predict whether a student will complete an online course.
- Predict the final performance score of a student.
- Compare different ensemble learning algorithms.
- Evaluate models using standard classification and regression metrics.
- Recommend the best-performing model.

---

## Problem Statement

Traditional machine learning models may produce unstable predictions. This project improves prediction performance by combining multiple models through ensemble learning techniques.

The project includes two machine learning tasks:

### Classification
Predict whether a student completes the course.

- 0 = Not Completed
- 1 = Completed

### Regression
Predict the student's final score ranging from 0 to 100.
